About WindCast

WindCast is a wind forecasting and decision-support platform designed for Khyber Pakhtunkhwa (KPK), Pakistan. It provides station-level forecast snapshots (Lead 1/3/6) and practical guidance for feasibility screening and reporting.

Project objective

The goal is to support early-stage wind energy assessment by combining forecasting outputs with a clear UI for: station selection, nearest-station mapping (GPS), forecast visualization, and exportable reports.

Key features

  • Station-based selection (10 PMD stations in KPK)
  • Nearest-station mapping using GPS for non-station locations
  • Multi-horizon forecast snapshot (Lead 1 / 3 / 6 months)
  • Interactive map + station heatmap layer
  • PDF report generation for documentation

Scope and limitations

WindCast uses station-level datasets. For locations beyond the station network, WindCast maps the user to the nearest station. This is a practical baseline approach; higher-resolution coverage can be added later using gridded wind datasets (e.g., reanalysis).

How to use

  1. Open the Dashboard and select a station (or use GPS to detect nearest station)
  2. Review forecast snapshots and guidance
  3. Open the Report page and export a PDF
